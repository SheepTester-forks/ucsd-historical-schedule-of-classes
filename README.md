# UCSD historical schedule of classes

UC San Diego's schedule of classes from Winter 1995 to Summer 2026.

Data were [scraped](https://github.com/SheepTester/uxdy/blob/main/scheduleofclasses/scrape.ts) from [Schedule of Classes](https://act.ucsd.edu/scheduleOfClasses/scheduleOfClassesStudent.htm).
As part of UCSD's TSS project, they're replacing a number of beloved but ancient applications with a third party vendor solution,
and Schedule of Classes is among [the web services being retired](https://docs.google.com/document/d/1fHj8FpxVfecCMlojX64OC6fLsoRJNTp4xzrLUsroZ_I/edit).
In case they do not preserve access to old schedule data, I'm archiving the schedules in this repo.
